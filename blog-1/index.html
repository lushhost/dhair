5 Windows Settings for Imporving Your Gaming Performance

Improve your Gaming Performance

Step 1:- Press Windows Key + R, Type "taskmgr", once taskmanager is open, go to startup option, disable all startup apps

Step 2:- Press Windows Key + R, Type "%temp%", once explorer is open, delete all files

Step 3:- Press Windows Key + R, Type "temp", once explorer is open, delete all files

Step 4:- Press Windows Key + R, Type "prefetch", once explorer is open, delete all files

Step 5:- Open Settings, Go to Gaming, Disable Xbox Game Bar, and Enable GameMode
